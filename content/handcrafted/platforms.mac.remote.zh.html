先讲人话版

🌐 Remote 这页，其实是在讲“让 Mac app 变成远程 Gateway 的遥控台”

你可以让 Mac 只当控制面板，而真正跑 OpenClaw 的机器在另一台主机上。

Mac 远程控制模式重点是 local / remote over SSH / remote direct查看官方原文

第一站

🚪 这里有三扇门：本地门、SSH 长隧道门、直连门

选哪一扇，决定数据怎么来回跑。

🏠

Local

所有东西都在这台 Mac 本地解决。

🧵

Remote over SSH

像拉一根保密长隧道，把远程 Gateway 端口悄悄接回本地。

🌍

Remote direct

直接连 ws/wss 地址，像走公开或 Tailnet 道路。

👀

区别

SSH 隧道模式下，Gateway 看见的可能是回环地址；直连模式才更容易看到真实客户端 IP。
