先讲人话版

Plugin inventory，是插件清单和分发方式的目录页

这页由 extensions/*/package.json、openclaw.plugin.json 和 npm package files 排除规则生成。它告诉维护者和用户：哪些插件在 core 包里，哪些是官方外部包，哪些只存在于源码 checkout。

生成型参考页重点看 Distribution column查看官方原文
Core npm package

已经包含在 openclaw npm 包里，不需要额外安装。

Official external package

由 OpenClaw 维护，但按需从 ClawHub 或 npm 安装。

Source checkout only

只在源码仓库里，不能当成公开可安装包来宣传。

使用方式

先看分发类型，再决定要不要 install 和 restart Gateway。
