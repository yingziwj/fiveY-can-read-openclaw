它不是花哨伴侣 app 路线，而是很扎实的服务端路线。

如果你想把 OpenClaw 放到 VPS、服务器、常开机器上，Linux 基本就是主场。

Linux Gateway 主场重点是 Node 24、onboard、SSH 端口转发查看官方原文

第一站

🚀 最快路径其实很朴素：装 Node、装 openclaw、跑 onboard

官方给的 VPS 初学者路线非常直接。

Node 24

像先把发动机装好。22 LTS 也能兼容，但首选 24。

npm i -g openclaw@latest

把 OpenClaw 全局装上，相当于把整套工具箱放进系统里。

openclaw onboard --install-daemon

像新手向导顺手帮你把常驻服务也铺好。

ssh -N -L 18789:127.0.0.1:18789 ...

这句像从你笔记本拉一根保密长线，把远程 Gateway 门口接回本地浏览器。
