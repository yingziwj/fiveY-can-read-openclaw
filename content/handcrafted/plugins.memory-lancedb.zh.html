先讲人话版

Memory LanceDB，像给 OpenClaw 放一只本地“记忆抽屉”

memory-lancedb 是官方外部记忆插件。它用 LanceDB 存长期记忆，再用 embeddings 帮 OpenClaw 在下一轮对话前找回相关内容。

它适合想把主动记忆放进本地向量数据库的人，但同一时间只能有一个插件拥有 active memory slot。

官方外部插件，需要安装关键配置：plugins.slots.memory = "memory-lancedb"查看官方原文

怎么理解

它做两件事：回想、记录

Recall

模型开口前，插件先从记忆库里找相近内容，像翻以前的课堂笔记。

Capture

模型回答后，它可以把重要事实写进记忆，像把新知识放回抽屉。

Embeddings

可以接 OpenAI-compatible embedding endpoint，也可以接本地 Ollama embedding。

边界

memory-wiki 这类伴随插件可以并排运行，但 active memory slot 只能由一个插件负责。
