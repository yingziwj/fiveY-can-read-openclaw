先讲人话版

🎪 Features 这页，其实就是 OpenClaw 的“本领大看板”

这页不是教程，也不是配置单。它更像你站在游乐园门口，看见一张大地图：这里能聊天、能接很多模型、能看图片和语音、能跑工具、还能装插件和自动化。

所以读这页的正确姿势，不是逐字背，而是先弄明白 OpenClaw 这套东西到底会哪几大类本领。

适合第一次想快速知道 OpenClaw 能做什么的人 这页重点是看全貌，不是抠细节参数 查看官方原文

第一站

🗺 先把这页看成一张“能力地图”

官方开头列了 Highlights。它不是在炫技，而是在告诉你：OpenClaw 不是只会聊天，它是一整套会接很多入口、会干很多动作的系统。

💬

Channels

它能接进很多聊天门口。Telegram、Discord、iMessage、WhatsApp 这些，都像不同的大门。

🔌

Plugins

本来没有的门和本领，还能靠插件往上加。像给乐高再拼新模块。

🧭

Routing

它不只是接消息，还会决定“这条消息该交给哪个代理处理”。

🖼

Media

不只文字。图片、音频、文档这些也能进，也能出。

第二站

🚪 第一大类本领：它能从很多地方“听见你”

原文“Channels”那段其实在说，OpenClaw 不被绑死在一个聊天软件里。它像会开很多门的门童。

📱 内建通道

WhatsApp、Telegram、Discord、iMessage 这些，是它自带就比较会走的门。

🧩 插件扩展通道

Mattermost、Matrix、Teams、Nostr 这些，像后来加装的新门。不是天生长着，但可以接上去。

👥 群聊也能玩

它不只会在私聊里回答，还能在群聊里按提及规则工作。像在教室里，被点名了再举手回答。

🛡 DM Safety 在说什么

意思是它不会默认谁来都理。pairing 和 allowlist 像门卫规则，先确认谁有资格进门。

第三站

🧠 第二大类本领：它不只会回话，还会分身、分路、分会话

原文“Agent”这一组，重点不是“有个 agent”这么简单，而是在讲它怎么把不同聊天分开处理。

🧰

内嵌运行时

OpenClaw 自己就带着代理运行时，不是只会转发消息。它收到消息以后，是真的会思考、调工具、再回话。

🛤

多代理路由

不同的人、不同账号、不同工作区，可以路由给不同代理。像总台接到电话后，知道该转给哪位同事。

🗂

会话隔离

私聊和群聊不会全糊成一锅。不同房间、不同来源，通常都有自己的会话格子。

🌊

长回复也能分段

它支持 streaming 和 chunking。翻成人话，就是边想边吐字，太长时还能拆成几块发，不至于一口气噎住。

第四站

🗝 第三大类本领：它能接很多“大脑供应商”

原文“Auth and providers”是在说，OpenClaw 不是只认一家模型。它像一个会带你去很多老师那里的领队。

🏪 35+ providers

意思不是一个神秘数字，而是在说可选大脑很多。OpenAI、Anthropic、Google，还有更多，都能接。

🪪 订阅登录也行

有些路线不是塞 API key，而是走 OAuth 或账号登录。像拿会员卡进门，不一定都是拿钥匙。

🏠 自建模型也行

如果你自己有 vLLM、SGLang、Ollama，或者兼容 OpenAI / Anthropic 的接口，也能接进来。像你自己家开的厨房，也能接上这套餐厅系统。

🎈 一句话收住

OpenClaw 更像“模型总插座”，不是只认某一家电器。

第五站

🖼 第四大类本领：不只看字，还会看图、听声音、碰文档

原文“Media”那段很短，但很重要。它说明 OpenClaw 不是只能处理纯文字的小机器人。

🖼 图片

你可以给它图，它也能回和图有关的内容。像你把一张照片递给它，它不只是装作没看见。

🎧 音频和语音

语音留言能转文字，文字也能变语音。它不只是“看字说字”，还能接住声音。

📄 文档

文件、文档也能进出。像你把一份纸质作业交给它，它能看，也能再还你处理过的版本。

📌 这节真正想告诉你

OpenClaw 跟现实世界打交道的入口，不只是一行文字消息。

第六站

🖥 第五大类本领：它还有自己的控制台、伴侣 App 和手机节点

原文“Apps and interfaces”这一段，是在说 OpenClaw 不只住在聊天窗口里。它还有自己的操作面板和设备触角。

🌐

WebChat / Control UI

像总控面板。不是纯聊天，而是能看、能配、能操作整套系统。

🍎

macOS 伴侣 App

像电脑边上的小助手窗口，随时能唤出来用。

📱

iOS / Android 节点

这不是简单的“手机聊天端”，而是能带上配对、语音、相机、位置、设备指令这些能力。

🧭

为什么这很重要

因为它说明 OpenClaw 不只是脑袋，它还有很多伸出去的手和眼睛。

第七站

🛠 第六大类本领：它真的会“做事”

原文最后的“Tools and automation”很关键。它说明 OpenClaw 不是光会聊天的娃娃，它能真的去执行动作。

🖥 Browser / exec

它能开浏览器、跑命令、在沙箱里做事。像不是只会出主意，还会真的去跑腿。

🔎 Web search

它能接很多搜索来源。像遇到不知道的题，不是瞎猜，而是会去图书馆和前台问。

⏰ Cron 和 heartbeat

它能按时间表做事。像不是非得你每次拍拍它，它才动，而是能按闹钟自己去执行固定工作。

📦 Skills / plugins / workflow

这些东西让它不只是“会做一件事”，而是能把做事方法、外接能力、流程编排都继续往上叠。

最后总结

🎈 把 Features 这页压成一句最好记的话

OpenClaw 不是一个单独的聊天机器人，而是一整套会接很多门、会用很多脑袋、会处理很多媒体、还能真的动手做事的系统。

如果你下一页只想继续看一页，我建议按你现在最关心的方向挑：想接聊天就去 /channels，想接模型就去 /providers/openai，想看它怎么动手就去 /tools。
