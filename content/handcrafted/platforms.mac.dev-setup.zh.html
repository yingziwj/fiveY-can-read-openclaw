第一站

🧰 先把两套工具箱准备好：Swift 那边的，和 Node 那边的

因为 Mac app 和 Gateway/CLI 不是同一种技术栈。

Xcode 26.2+

像搭 Swift 施工现场的主工具箱。

Node 24 + pnpm

这是另一边的工具箱，给 Gateway、CLI、打包脚本用。

pnpm install

先把整个仓库的零件都装齐。

./scripts/package-mac-app.sh

像按下“开始拼装并打包成 OpenClaw.app”的大按钮。
