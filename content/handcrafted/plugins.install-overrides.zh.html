先讲人话版

Plugin install overrides，是维护者测试安装流程时用的临时换轨

它允许维护者把 setup-time plugin install 指向指定 npm 包或本地 npm pack tarball。普通用户仍应使用 openclaw plugins install。

仅用于 E2E 和包验证覆盖来源会执行插件代码查看官方原文
什么时候用

发版前想验证某个 npm 版本或本地 tarball 的安装行为。

什么时候不用

普通安装、日常使用、生产机器上不该靠 override。

安全边界

只在隔离 state directory 或一次性测试机器里使用。

格式

override map 以 plugin id 为 key，值可以是 npm: spec 或 npm-pack: tarball。
