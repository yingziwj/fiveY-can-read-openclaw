先讲人话版

🧭 Provider Directory 这页，其实是在帮你先决定“去哪家模型商店买脑袋”

OpenClaw 能接很多 provider。这页不是让你背名单，而是帮你先分清：你是想走最省事的主流路线，还是已经心里有指定店家。

所以它更像导览图。先选商店，再看进门方式，最后才是货架上的模型。

适合第一次挑模型提供商的人 先选店家，再看认证和模型 查看官方原文

第一站

🏪 把 providers 想成一排模型商店

OpenAI、Anthropic、Google、OpenRouter、Ollama，这些名字本质上都是不同店家。OpenClaw 只是负责带你进去买到回答。

☁️ 云上商店

像 OpenAI、Anthropic、Google、Bedrock 这种，通常是拿 API key 或账号门票去远程调用。

🛒 聚合商店

像 OpenRouter、LiteLLM、Cloudflare AI Gateway，更像一个大市场，一家门面后面连着很多摊位。

🏠 本地商店

像 Ollama、vLLM、SGLang，更像把模型搬回自己家，隐私和控制权更强。

🎈 一句话

Provider 不是模型本身，而是模型住在哪家店、你怎么进去。

第二站

🚶 第一轮别贪多，先选一条最短路

Quick start 的用意很朴素：第一次先接通一条最稳的线，再考虑扩展。

最省心

OpenAI 或 Anthropic。文档全、社区熟、踩坑资料多。

想多试几家

OpenRouter。一个入口试很多模型，适合比较和试味道。

想放自己家里

Ollama。适合本地模型、隐私优先和离线玩法。

企业云

Bedrock、Vertex、Azure 这类更适合已经在云平台里有组织规则的人。

最后总结

🎈 把 Provider Directory 压成一句最好记的话

这页不是模型百科，而是选店指南。先决定去哪家店，再去看那家店的门票、货架和规矩。

如果你接下来只继续看一页，我建议先读 /providers/models。
