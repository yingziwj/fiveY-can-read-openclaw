先讲人话版

🍎 macOS 这页，其实是在讲“OpenClaw 在 Mac 上不只是一个窗口，而是一位有权限、有菜单栏、有 node 能力的总助手”

它既能管理本地 Gateway，也能连远程 Gateway，还能把 Mac 自己的能力暴露给 agent。

macOS 伴侣 app重点是 menu bar、Gateway broker、local/remote 两种模式查看官方原文

第一站

🧩 它最像“菜单栏总控台 + 权限代理 + Mac 节点外设”三合一

不是单纯聊天窗，也不是只会开 Gateway 的启动器。

🔔

菜单栏状态

能看通知、健康、工作状态。

🔐

TCC 权限拥有者

屏幕录制、麦克风、自动化这些 Mac 权限都得由它来扛。

🤖

Mac 节点能力

Canvas、Camera、system.run 这些 Mac 特有能力都能暴露给 agent。

🌉

Gateway broker

本地模式下管 launchd，远程模式下管 SSH / Tailscale 这类连接桥。
