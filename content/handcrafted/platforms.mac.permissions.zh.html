先讲人话版

🔐 Permissions 这页，其实是在讲“为什么 Mac 权限会突然失忆，以及怎样别把它惹毛”

macOS 的 TCC 很记仇：签名、bundle id、安装路径，只要你换了，它就可能把你当成陌生新 app。

macOS TCC 权限重点是固定路径、稳定签名、bundle id 一致查看官方原文

第一站

🪪 TCC 认的是“你是谁、你住哪儿、你签名是不是同一个人”

三样里随便换一样，系统都可能把你当新人。

📍

固定路径

最好一直从同一个 dist/OpenClaw.app 路径跑。

🏷️

固定 bundle id

换 bundle id 就像改身份证号，旧授权不一定还认你。

✍️

稳定签名

ad-hoc 签名最容易让权限记忆丢光，不适合拿来测试权限持久性。

♻️

失忆时恢复

删旧授权、重启 app、必要时用 tccutil reset 清表再来。
