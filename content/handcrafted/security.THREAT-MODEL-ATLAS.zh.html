先讲人话版

🧭 Threat Model v1.0 这页，就是“把 OpenClaw 可能遇到的坏事，按 ATLAS 地图一格一格摆出来”

它不是在吓人，而是在给攻击故事贴标签。这样你就知道威胁从哪里来、会打到哪里、应该怎么防。

适合想看完整威胁地图的人 先看框架，再看范围，再看威胁编号和贡献方式 查看官方原文

第一站

🗺 先认地图名：ATLAS 是 AI/ML 版安全地图

官方把 MITRE ATLAS 放在标题里，是因为这张地图就是给 AI 系统的攻击路径用的。

Framework

像一个统一坐标系，攻击故事都往这里放。

Version / last updated

像地图印刷版号，告诉你这张图是什么时候定稿的。

Data flow diagrams

像把路线上每个门、每个窗口、每个回路都画出来。

第三站

🏷 威胁编号和类别，就是给坏事贴标签

页面里 `T-EXEC-003` 这种编号，不是摆造型，是为了让同一件坏事可以被反复引用。

RECON / ACCESS / EXEC

像“先看、再进、再动手”的攻击节奏。

PERSIST / EVADE / DISC

像进来后留着不走、躲着不被抓、偷偷探路。

EXFIL / IMPACT

像把东西带走，或者直接造成破坏。

最后总结

🎈 这页的真正用途，是让安全讨论有统一地图

你不是在背 ATLAS 术语，而是在学会：同一个攻击故事，应该被放在哪个坐标里。
