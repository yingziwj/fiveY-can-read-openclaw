先讲人话版

🤖 Android 这页，其实是在讲“Android 目前更像一台 node 设备，不是 Gateway 主机”

它负责连到别处的 Gateway，然后贡献手机这边的能力。

所以重点不是“在 Android 上独立开整套 OpenClaw”，而是“把手机接进现有系统里”。

Android node重点是配对、WebSocket、源码自建查看官方原文

第一站

📦 它还没公开发版，所以现在更像开发者自己打包的试验机

官方已经把源码放出来了，但不是那种去应用商店一键安装的状态。

源码自建

要自己用 Java 17 和 Android SDK 把 app 编出来，像自己从零拼一台设备。

需要外部 Gateway

Android 不托管 Gateway，它只负责连过去。

配对

连上 WebSocket 后，还是要走 device pairing，先拿到 node 工牌。

🎈 最后记一句

Android 现在是陪跑 node，不是单机大本营。
