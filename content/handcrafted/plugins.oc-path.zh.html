先讲人话版

OC Path plugin，是给工作区文件叶子节点准备的精确小镊子

oc:// 地址指向 workspace 文件里的某个叶子或一组叶子。这个插件提供 openclaw path CLI，让工具和编辑器不用自己写解析器，也能稳定读写 YAML、Markdown、JSONC 和 JSONL 的局部内容。

bundled 但 opt-in源码位置：extensions/oc-path/查看官方原文
像小镊子

它不是整文件搬运，而是精准夹起某个字段、段落、记录或 frontmatter。

保留格式

JSONC 注释、Markdown 结构、YAML 文档都尽量按原样往返。

谁会用

自托管者、编辑器扩展、agents 和 hooks 都可以把它当作确定性底座。

保护

统一走 redaction sentinel guard，避免敏感内容被随手写坏。
