先讲人话版

🥧 Raspberry Pi 这页，其实是在教你“把 OpenClaw 养在家里一台一直开着的小盒子上”

这条路很可爱，也很实用。因为 Pi 不负责本地跑大模型，它更像一位常驻门卫，负责挂着 Gateway、接通道、守着自动化任务。

所以官方最想告诉你的不是“Pi 很强”，而是“只要你把系统、Node、存储和散热照顾好，它完全够当一个长期在线的小值班员”。

适合家庭常驻小主机 重点是 64 位系统、Node 24、swap、SSH 隧道 查看官方原文

第一站

🧺 先把小盒子的生活用品备齐

官方先列前置条件，不是在凑字数，而是在防止你后面装到一半才发现家当太少。

🧠

Pi 4 / 5，2GB 起步

像这位小门卫至少得有张像样的桌子，4GB 会更舒服。

💽

存储别太抠

SD 卡能用，但 USB SSD 更像正经衣柜，速度和寿命都更好。

64

必须 64 位系统

这一条别糊弄。32 位就像给它穿错鞋，后面很多东西会别扭。

🔌

稳定供电

小盒子最怕半夜饿晕，电源不稳会让你以为是软件坏了。

第二站

🧭 Setup 那一长段，其实是在做“给小门卫穿衣、配钥匙、安排宿舍”

一条条看就不乱了。

💿 刷 Raspberry Pi OS Lite 64-bit

像先给它穿最轻便、最适合值班的制服，不带桌面，少吃资源。

ssh user@gateway-host

像你第一次从自己的电脑走到这台小门卫的值班室门口。

sudo apt update && sudo apt upgrade -y

像先把旧家具和旧螺丝全检查一遍，别带着过期零件开始上班。

curl -fsSL https://deb.nodesource.com/setup_24.x | sudo -E bash -

像先把 Node 24 这台发动机的安装向导请进来。

第三站

🛏️ swap 那段，翻成人话就是“给记忆力不大的小门卫铺一张临时床”

尤其 2GB 设备，这段很重要。

sudo fallocate -l 2G /swapfile

像先腾出一块临时休息垫，内存不够时能顶一下。

sudo swapon /swapfile

像正式把这张备用床铺投入使用。

vm.swappiness=10

像告诉系统：“不到真需要的时候，别太依赖这张临时床。”

📌 官方为什么特地点名这段

不是为了显得专业，而是因为小内存机器最容易在安装和构建时突然趴下。

第四站

🚀 安装 OpenClaw 后，再用 SSH 隧道把控制台牵回你自己的电脑上

这就是 Pi 页里最有画面感的一段。

curl -fsSL https://openclaw.ai/install.sh | bash

像把官方安装车直接开进 Pi 的值班室。

openclaw onboard --install-daemon

像搬完家具后，顺手把后台值班制度也装好。

ssh user@gateway-host 'openclaw dashboard --no-open'

像先让 Pi 报一下：“我的控制台门牌是多少？”

ssh -N -L 18789:127.0.0.1:18789 user@gateway-host

像拉一根私人电话线，把 Pi 里的控制台窗口牵回你眼前这台电脑。

第五站

🩺 Performance 和 troubleshooting，都在教你怎么照顾这位小门卫别累坏

Pi 路线要稳定，养护比炫参数更重要。

💽 用 USB SSD

像把纸箱换成真正抽屉，系统和日志都会轻松很多。

🧠 compile cache

像给它准备一个常用动作小抄本，下次启动不用再从头背。

🌡️ 看 throttling

如果它太热、被降速，就像门卫热得直喘气，工作自然慢。

🎈 收尾一句

Raspberry Pi 路线的重点不是堆性能，而是把这位小值班员养得稳定、长久在线。
