先讲人话版

Admin HTTP RPC，是给可信自动化开的 Gateway 控制台小门

这个 bundled 插件会把部分 Gateway control-plane 方法暴露成 HTTP。它默认关闭，只适合私有主机工具、tailnet 自动化或可信内网入口。

不要把它直接暴露到公网。通过认证的调用者可以执行 allowlist 里的管理方法。

默认关闭路由：POST /api/v1/admin/rpc查看官方原文
什么时候用

普通 WebSocket RPC 客户端不方便，但可信主机自动化需要控制 Gateway 时。

风险

这是操作员控制面，不是给公众请求玩的普通 API。

启用后

它和 Gateway 用同一个 listener，只新增受控 HTTP route。

检查

只在私有网络里启用，并确认 HTTP auth 和 allowlist 都符合预期。
