先讲人话版

🚉 Bundled Gateway 这页，其实是在讲“Mac app 现在不再自带 Gateway 发动机，而是要求你先装外部 openclaw CLI”

也就是说，App 自己不是发动机本体，它更像一个调度台，去管理外面的 Gateway 服务。

外置 Gateway 方案重点是外部 CLI、launchd、local mode查看官方原文

第一站

🧰 先装 CLI，就像先把发动机装进 Mac

App 自己不会再把 Node/Bun/Gateway 打包带着跑。

npm install -g openclaw@<version>

这句就是把 Gateway 发动机装进系统里。

Install CLI 按钮

App 里的按钮，本质上也是在帮你跑同样的全局安装动作。

LaunchAgent

装好后会由 launchd 负责常驻，就像把发动机挂进系统后台值班表。

🎈 最后记一句

现在的 Mac app 是调度员，不是自带引擎的一体机。
